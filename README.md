# 🚗 RapidRide Taxi Management System

**SIMAD University** | Faculty of Computing | PL/SQL Programming Course

---

## 📋 Project Overview

RapidRide is a comprehensive ride-hailing and taxi management system that connects customers with drivers, manages ride bookings, handles payments, and maintains detailed records of all transactions.
